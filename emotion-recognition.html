Emotion Recognition System

Real-Time Facial Emotion Detection using Deep Learning

Project Overview

Designed a real-time emotion recognition system using Convolutional Neural Networks (CNN). This project implements OpenCV for facial detection and Keras for deep learning model training, enabling accurate emotion classification from live video feeds and images.

Key Features

  • Real-time emotion detection from webcam feed
  • Multiple CNN architectures comparison (DenseNet, ResNet)
  • OpenCV-based facial detection and preprocessing
  • High accuracy emotion classification (7 emotions)
  • Live video processing and annotation
  • Batch image processing capabilities
  • Model performance evaluation and comparison

Emotion Categories

  • Happy
  • Sad
  • Angry
  • Surprised
  • Fearful
  • Disgusted
  • Neutral

Research Focus

This project involved comprehensive research on different CNN architectures including DenseNet and ResNet to determine the most effective approach for facial emotion recognition. The system achieves high accuracy through advanced preprocessing techniques and optimized neural network architectures.

Technologies Used

CNN OpenCV Keras DenseNet ResNet Python TensorFlow NumPy
