Conversational Sales Agent

Multi-Agent AI System for Automated Sales Interactions

Project Overview

Developed an AI-powered sales agent using Google's Agent Development Kit to handle concurrent lead interactions via a sophisticated multi-agentic architecture with session persistence and advanced context tracking. This system revolutionizes sales automation through intelligent conversation management.

Key Features

  • Multi-agent architecture for concurrent lead handling
  • Session persistence and context tracking
  • Google's Agent Development Kit integration
  • LangGraph workflow orchestration
  • RAG-based knowledge retrieval
  • FAISS vector database for efficient search
  • Real-time conversation state management
  • Intelligent lead qualification and routing

Multi-Agent Architecture

  • Lead Qualifier Agent: Initial customer assessment and categorization
  • Product Specialist Agent: Detailed product information and recommendations
  • Pricing Agent: Quote generation and negotiation handling
  • Follow-up Agent: Automated follow-up and nurturing campaigns
  • Escalation Agent: Human handoff for complex queries

Technical Innovation

This project showcases advanced AI orchestration using LangGraph for complex workflow management. The system maintains conversation context across multiple interactions, enabling personalized and coherent sales experiences. The integration of FAISS ensures rapid information retrieval for product queries and customer history.

Technologies Used

Google ADK LangGraph Multi-Agent RAG FAISS Python LangChain Vector DB
